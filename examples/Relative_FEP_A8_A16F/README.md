# Relative EE A8 A16F

## This folder contains all the files to run unoptimized expanded ensemble simulations on the A8 A16F mutation system. Additionally it contains files to run pylambdaopt and plot free energy differences.


**Simulation_Files**: Contains files used to run EE simulations.

**get_dG_and_WL.py** : Python script to extract dG and log10 WL values from GROMACS log files. Outputs are the Numpy files in folder.

**EE_1_dhdl.xvg**: Initial DHDL file used for optimiation.
